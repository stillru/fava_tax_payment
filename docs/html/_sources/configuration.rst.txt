Configuration
=============

Tax Configuration
-----------------

The extension creates a ``config/tax_config.json`` file in your working directory on first run.
You can customize this file with your payment details either by directly editing it or using
the web UI provided in the Fava interface.

Configuration Format
--------------------

.. code-block:: json

   {
     "payer": {
       "name": "Your Name or Company",
       "account": "Your-Account-Number"
     },
     "expense_category": "Expenses:Taxes",
     "taxes": {
       "income": {
         "purpose": "Income Tax Payment",
         "recipient": "Tax Authority Name",
         "recipient_account": "Tax-Authority-Account",
         "payment_code": "Code",
         "model": "Model",
         "reference_number": "Reference"
       }
     }
   }

Web Configuration UI
--------------------

The extension provides a web-based configuration interface within Fava. To use it:

1. Navigate to the "Tax Payment PDFs" section in Fava
2. Scroll down to the "Edit Configuration" section
3. Modify the settings as needed
4. Click "Save Configuration" to apply your changes
